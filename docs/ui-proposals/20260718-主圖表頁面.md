## 方案 A：深色交易終端風（TradingView 風格）

- **背景**：`#131722`（深灰藍）、面板 `#1E222D`、格線 `#2A2E39`
- **文字**：`#D1D4DC`、次要 `#787B86`
- **K 線**：台式紅漲綠跌 — 漲 `#EF5350` 紅、跌 `#26A69A` 綠（可切美式）
- **ZigZag**：`#FFB74D` 橘黃折線；波浪標記白字圓底
- **乖離率**：`#42A5F5` 藍曲線，極端帶半透明紅/綠色塊
- **排版邏輯**：全螢幕沉浸式，資訊密度高，適合長時間盯盤閱讀，減少眼睛疲勞

## 方案 B：明亮極簡儀表板風

- **背景**：`#F8F9FA` 淺灰、卡片 `#FFFFFF` 白 + 陰影圓角
- **文字**：`#212529`、次要 `#868E96`
- **K 線**：漲 `#E03131` 紅、跌 `#2F9E44` 綠
- **ZigZag**：`#5F3DC4` 深紫折線；波浪標記深紫字
- **乖離率**：`#1971C2` 藍曲線，極端帶淺色背景
- **排版邏輯**：卡片式分區（主圖一張卡、副圖一張卡），留白多、清爽，適合白天瀏覽與分享截圖

## 方案 C：深藍金融資訊風（Bloomberg 風格）

- **背景**：`#0A1929` 深藍、面板 `#102A43`
- **文字**：`#F0F4F8`、強調 `#F7C948` 金黃
- **K 線**：漲 `#F76707` 橘紅、跌 `#20C997` 青綠
- **ZigZag**：`#F7C948` 金色折線；波浪標記金字
- **乖離率**：`#63B3ED` 亮藍曲線
- **排版邏輯**：金融資訊終端氛圍，金色強調關鍵數字（最新乖離率、最新轉折價），Header 常駐關鍵指標摘要列
